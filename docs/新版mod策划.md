## 二、兽装品质评定

### 2.1 品质计算公式
成品兽装品质 = (装师等级系数 × 兽装品质值) × (1 + 装师阶段加成)

**装师等级系数**
装师等级系数 = 装师当前等级 / 装师满级等级 × 100%



**兽装品质值**
兽装品质值 = 材料品质参数 × UI互动参数 × 部位品质固定值



### 2.2 部位品质固定值
| 部位 | 固定值 |
| :--- | :--- |
| 头 | 175 |
| 身体 | 165 |
| 手爪 | 100 |
| 腿 | 165（曲腿+5） |
| 尾巴 | 95 |
| 脚 | 100 |

### 2.3 UI互动参数
UI互动参数 = 玩家获得的所有UI互动分数 / 所有UI互动的总分数 × 100%



### 2.4 材料品质参数
总参数 = 1 × 各项系数的乘积 × 填充混搭程度



**各项系数**：
- 海绵头骨 = ×1
- 树脂头骨 = ×0.6
- 粘液球粘合 = ×1
- 树脂粘合 = ×0.5

**填充混搭程度**
填充混搭程度 = 1 - (占比最大填充数 / 总填充数) × (使用填充种类数 / 2)



### 2.5 装师阶段加成
| 阶段 | 加成 |
| :--- | :--- |
| 新手装师 | -2% |
| 初级装师 | 0% |
| 熟练装师 | +2% |
| 老手装师 | +4% |
| 满级 | +8% |

### 2.6 品质分阶
品质分阶比例 = (成品兽装品质 / 极限品质) × 100%



| 阶位 | 比例范围 |
| :--- | :--- |
| D | 0 ~ 1.25% |
| C | 1.26% ~ 13.75% |
| B | 13.76% ~ 26.25% |
| A | 26.26% ~ 51.25% |
| S | 51.26% ~ 95% |
| S+ | 95.01% ~ 99.99% |
| S++ | ≥ 100% |

### 2.7 部位合成品质继承
> 部位合成时，合成品取前合成物品质分阶比例均值并重新评定品质。
>
> **示例**：0%（D）爪 + 100%（S++）身 = 50%（A）身包含爪

### 2.8 品质加成效果
| 阶位 | 加成效果 |
| :--- | :--- |
| D | 减益 |
| C ~ B | 无 |
| A ~ S | 少量增益 |
| S+ ~ S++ | 较多增益 |

## 三、装师等级制度

### 3.1 阶段与等级
| 阶段 | 等级范围 |
| :--- | :--- |
| 新手装师 | 1 ~ 10级 |
| 初级装师 | 11 ~ 20级 |
| 熟练装师 | 21 ~ 40级 |
| 老手装师 | 41 ~ 80级 |

### 3.2 等级提升经验获取

**1. 制作部位获得经验**
经验 = 材料品质参数 × UI互动参数 × 兽装固有品质值



**2. 接委托获得经验**（20级以后才能接）
经验 = 接委托所需级数 × 部位品质固定值 / 100



**3. 升级所需经验**
升到下一级的经验 = 100 × (0.5 × (当前等级 + 1))

## 五、兽装穿戴效果

> **注**：潮湿度与脏污值若各部件不同，则穿上后均摊。

### 5.1 高温与中暑

**判定条件**：所处位置温度 > 1.2 时，每 `(1 + 潮湿度/100)` 秒有 `(过热常数 × 所处温度)`% 的概率使"中暑"效果上升1等级。

**过热常数**：
| 部位/材质 | 过热常数 |
| :--- | :--- |
| 兽头（海绵头骨） | +0.03 |
| 兽头（树脂头骨） | +0.02 |
| 身体 | +0.02 |
| 伪全身体 | +0.01 |
| 曲腿 | +0.03 |
| 直腿 | +0.02 |
| 兽爪 | +0.005 |
| 兽装脚 | +0.01 |
| 尾巴 | +0.005 |

**中暑效果**：
- **1~2级**：仅视觉效果，逐步加深
- **3~4级**：每2秒有 `(当前等级/100)` 概率获得反胃效果5秒
- **>4级**：禁止疾跑，扣除 `(当前等级)` 血量上限（最多扣至5心），每15秒有 `(当前等级/100)` 概率扣除 `(当前等级/2)` 血量

**中暑消退**：
- 当过热常数 ≥ 0 时：右键使用兽装风扇，每秒有 `(0.5 - 过热常数)/10` 概率消退1等级
- 当过热常数 = 0 时：
    - 空气中：每分钟消退1等级
    - 水中：每40秒消退1等级
    - 饮用水瓶：25%概率直接消退1等级
    - 所处环境温度 < 0：每分钟额外消退1等级

### 5.2 脏污

**累计方式**：
- 自然累计：+0.1/秒
- 受伤时：+5/每1点伤害
- 中暑时：额外增加 `0.05 × 中暑等级`/秒

**清洁方式**：
| 方式 | 详情 |
| :--- | :--- |
| 潮湿自洁 | 潮湿值 ≥ 25 时，每点潮湿值每tick -0.005脏污值，至多减到500 |
| 燃烧清洁 | 燃烧受伤每次中量磨损（-8%耐久）并 -10%脏污（地狱-15%），脏污值低于150时直接清零 |
| 小清洗 | 扣除500脏污值，耗时35秒 |
| 大清洗 | 扣除1500脏污值，耗时75秒 |

**脏污效果**：
| 脏污值范围 | 效果 |
| :--- | :--- |
| 0 ~ 500 | 无影响 |
| 500 ~ 1500 | 出现视野肮脏效果 |
| 1500 ~ 3000 | 视野效果加深，略微加剧磨损（每次35%概率磨损×2） |
| > 3000 | 视觉效果最大，严重加剧磨损（75%×2，25%×3），恶心（每15秒有 `(脏污值/10000)` 概率中毒1级4秒，`(脏污值/50000)` 概率2级中毒2秒 + 反胃5秒） |

### 5.3 潮湿

**累消规则**：
| 环境 | 潮湿度变化 |
| :--- | :--- |
| 空气中 | 每秒减少 `(0.8 - 群系降水) × [群系温度（<0时取0）]` |
| 海洋群系 | 每秒增加0.3 |
| 雨/雪中 | 每秒增加1/0.8 |
| 水中（未浸没） | 每秒增加5 |
| 水中（浸没） | 每秒增加8 |
| 晾衣架上（群系降雨 ≤ 0.8） | 每秒减少 `(0.8 - 群系降水) × [群系温度（<0时取0）] × 3` |
| 晾衣架旁有火/营火/岩浆 | 每秒额外减少0.5 |

**潮湿效果**：
- 潮湿度 ≥ 25 时：每点潮湿度每tick -0.005脏污值，至多减到500
- 每点潮湿度增加0.01秒中暑判定间隔
- 每点潮湿度增加质量0.07

## 六、兽装特性系统

### 6.1 特性综述
- 每件兽装制作时会随机携带特性。
- 可通过**特性结晶拓展槽**进行增减。
- **分类**：正面特性、负面特性、特殊特性。

### 6.2 特性结晶槽位
| 品质阶位 | 槽位数 |
| :--- | :--- |
| D ~ C | 0格 |
| B | 1格 |
| A | 2格 |
| S ~ S++ | 3格 |
| **整体** | 均为1格 |

### 6.3 特性结晶镶嵌规则

**仅可镶在部位**：
轻装上阵、盔甲兼容、内置风扇、耐火涂层、宽阔视野、微光视野、心之容器、利爪、长爪、迅捷、强力跳跃、突围、额外存储、轻功、多段跳、肉垫缓冲、弹簧锁

**仅可镶嵌在整体**：
耐脏技术、防水布料、软毛缓冲、复生、月光之力、瞬间失忆

**均可镶嵌**：
潜能无限、结晶稳定

**【负面消除】结晶**：
- 一手拿结晶，一手拿需要消除负面特性的部位，Shift + 右键
- 按由上至下顺序消除1个负面特性，结晶消失
- 无负面特性时不可使用

### 6.4 特性结晶的合成与升级
- **空白结晶**：工作台合成
- **部分结晶**：工作台 + 空白结晶
- **大部分结晶**：结晶台 + 空白结晶
- **升级**：所有结晶均需在结晶台进行

### 6.5 特性结晶的镶嵌操作
- **部位镶嵌**：在兽装工作站中，将特性结晶镶嵌至各部位槽位
- **整体镶嵌**：需要四件整体镶嵌情况一致的兽装部分才可共同进行

### 6.6 特性的获得与继承
- 镶嵌未损坏的特性结晶或制作时，兽装获得特性
- 除【潜能无限、瞬间失忆】外，相同种类结晶无法重复镶嵌
- 与制作特性相同的结晶可镶嵌，取等级更高者体现（不覆盖制作特性）
- **组件合成时**（如爪+身=身包含爪），后者继承前者的制作属性，并保留前者的特性结晶（若槽位不足则掉落）

### 6.7 特性结晶的损坏与修复
- **耐久**：每个特性结晶拥有2048耐久
- **消耗速度**：结晶等级 > 制作等级时，以 `0.4/秒` 的速度减少
- **【结晶稳定】**：满级时减缓至 `0.1/秒`
- **损坏后果**：耐久耗尽变为损坏结晶，等级-1（最低至1）
- **修复**：在结晶台中利用对应材料修复，恢复100%耐久（不恢复等级）

## 七、特性详细列表

### 7.1 负面特性

#### 1. 失败制作
- **至高等级**：1级
- **是否可合成结晶**：否
- **描述**：这是你起始的造物，不知道你用了多长时间来完成它。虽然毛毛剃的狼牙狗啃，造型也歪七扭八的，但是这是你第一次尝试让ta活过来。
- **效果**：随着玩家行动（奔跑、跳跃、攻击等），兽装有约1%概率从身上脱落。
- **获得**：新手装师阶段且UI互动总得分 < 50%时，有 `(1 - 得分比例)` 概率获得。

#### 2. 易损
- **至高等级**：1级
- **是否可合成结晶**：否
- **描述**：emmmm......你这家伙缝的也太烂了点吧，你看这块布都快掉下来了。
- **效果**：兽装耐久上限降低10%~30%（随机）。
- **获得**：新手装师阶段且UI互动总得分 < 50%时，有 `(1 - 得分比例)` 概率获得。

---

### 7.2 正面特性

#### 1. 轻装上阵
- **至高等级**：5级
- **获取方式**：特性结晶 / 优质制作
- **描述**：减轻结构重量的办法有两种，魔法和优化材料。不过用哪个方法不重要，谁都会喜欢一套轻一些的兽装吧。
- **效果**：每级使基础负重-1（至多至0）。
- **获得**：制作品阶A~S级时，有 `(得分比例/8)` 概率获得1级；S+及以上时另有 `(得分比例/8)` 概率升级为2级。

## 十一、接单、委托他人与交易

### 11.1 接单

**刷新机制**：装师手册拥有接稿拓展时，每隔一段时间随机刷新委托需求。

**委托需求内容**：
等级限制、设计图、需求制作部位、品质底线、预计报酬、时间限制

**接取流程**：
1. 在栏位内放入空白设计图纸
2. 复制委托设计图到空白图纸
3. 获得预计报酬的25%

**预计报酬**：随等级限制、设计图复杂度、品质底线、需求部位增加而增加，随时间限制减少而增加

**接取上限**：`(当前等级 / 10)` 向下取整

**交付**：
- 在手册已接取委托栏目中，放置对应设计图和完成的符合要求的部位
- 交付后获得剩余报酬和装师经验
- 若优质制作超过品质底线，可获得额外打赏：`(实际品质比例 / 品质底线比例 × 预计报酬 × 0.5)`

**放弃**：
- 放置对应设计图和预计报酬的25%即可放弃
- 超时视为自动放弃，需偿还预计报酬的50%才能消失
- 全部还款前不可继续接取其他委托

### 11.2 委托他人

**材料**：设计图纸 + 空白设计图纸 + 纸 = 委托单（不消耗原设计图纸）

**流程**：
1. 右键委托单，选择需求
2. 放入预计报酬
3. 选择委托的玩家，选定后消耗
4. 被委托方会在其手册上刷新玩家委托（高亮+异色）

**委托失败**：超时/正常放弃时，预计报酬返还委托方；制作超时多惩罚的25%不返还

## 十三、重要方块UI

### 13.1 兽装工作站

**主页面选项**（按钮融入背景，鼠标移上显示文字+白边）：
兽装设计、兽装制作、兽装修复、兽装清洗、结晶镶嵌、毛布存储

**兽装设计**：
- 上方放置设计图纸位置，右面显示设计预览
- 左侧为更改选项（切换预览部位、花纹选择、颜色选择）
- 右下确认/取消，左上退出

**兽装制作**（子选项）：兽头、兽爪、兽身、兽腿、尾巴、兽脚

**兽头制作**：
- 左旁：头骨雕刻（雕刻刀+头骨材料）和舌头制作（填充材料）
- 中间：针线盒位置
- 右面：头骨、牙齿、舌头、眼睛、黏胶位置
- 下方：开始制作

**兽爪制作**：
- 左上：版型绘制（纸、羽毛、墨囊）
- 左下：肉垫缝制（填充、掌/指选择）
- 中间：针线盒
- 右面：兽爪整体制作（版型、指肉垫、掌肉垫、指甲、填充）
- 下方：开始

**兽身制作**：身DTD、针线盒、填充位，下开始

**兽腿制作**：腿DTD、针线盒、填充位，下开始

**尾巴制作**：左版型绘制，中针线盒，右整体制作（版型、填充），下开始

**兽装修复**：上设计图纸，中修复部位槽+针线盒，下开始

**兽装清洗**：上全套兽装槽，中水桶×2、刷子、清洁剂、毛刷，下选择清洗方式与开始

**结晶镶嵌**：
- 左全套兽装槽，放入后延伸出结晶拓展槽
- 右整体拓展槽，未放全不开放使用

**毛布存储**：显示所有颜色毛布槽位，每色仅可填充1块；所需毛布不足时，相关操作被禁止
